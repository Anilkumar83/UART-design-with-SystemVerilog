 UART Protocol Design in SystemVerilog 🎉
 
 🔹 Project Highlights:
 
 ✅ Implemented UART Transmitter & Receiver modules with baud rate generator.
 
 ✅ Added parity handling, start/stop bits, error detection (parity & framing errors).
 
 ✅ Built a UVM-like testbench for self-checking verification.
 
 ✅ Verified corner cases through error injection and noise simulation.
 
 ✅ Simulated and tested on Vivado (or ModelSim) for functional correctness.
